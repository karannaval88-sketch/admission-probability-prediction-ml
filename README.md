# 🎓 Admission Probability Prediction (Machine Learning)

## 📌 Overview
This project predicts the probability of a student getting admission into graduate programs based on academic and profile factors. It uses regression techniques to estimate admission chances.

## 📊 Dataset
The dataset includes the following features:
- GRE Score
- TOEFL Score
- University Rating
- SOP (Statement of Purpose strength)
- LOR (Letter of Recommendation strength)
- CGPA
- Research Experience

## ⚙️ Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib / Seaborn
- Scikit-learn

## 🚀 Workflow
1. Data Cleaning and Preprocessing  
2. Exploratory Data Analysis (EDA)  
3. Feature Selection and Scaling  
4. Model Training (Linear Regression)  
5. Model Evaluation  

## 📈 Results
- Achieved good prediction accuracy using regression models  
- Evaluated using:
  - R² Score  
  - Mean Squared Error (MSE)

## 📷 Visualizations
- Correlation Heatmap  
- Feature vs Target Graphs
